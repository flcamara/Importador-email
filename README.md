# Importador de Email ✉️

Projeto que consiste em uma importação de arquivos CSV e envio massivo de e-mails.

## Pré-requisitos ⚠️

NodeJS >= 18.17.0

Certifique-se de criar seu .env com as informações necessárias.

## Instalação 🖥️

```bash
npm install

npm run start
````
